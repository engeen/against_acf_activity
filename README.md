--------------------
Почему тут:
1) легко внести свое имя всписок
2) авторизацию выполняет github
3) персданные не собираются (только email и ваше имя / строчка о себе публично будут доступны) 
4) ботам сюда будет попасть сложно: у Github очень хорошая защита от них
5) технологическая "неподделываемость"

---------------------
Как подписать (инструкция для НЕ-айтишников)

1. Зарегистрируйтесь на GitHub
2. Нажмите кнопку с карандашом в правом верхнем углу чтобы отредактировать этот файл 
3. <img width="1371" alt="Screenshot 2024-10-28 at 16 45 32" src="https://github.com/user-attachments/assets/4336f4b3-a5dc-4884-b580-5332a0e47af6">
4. Нажмите Fork this repository
5. Отредактируйте файл, внеся ваше имя/описание в список ниже
6. Нажмите Commit Changes в правом верхнем углу
7. А потом просто со всем соглашайтесь и говорите да:
- propose changes
  <img width="606" alt="Screenshot 2024-10-28 at 16 46 07" src="https://github.com/user-attachments/assets/87a0af25-03f2-4387-89be-b8c7b3209f34">
